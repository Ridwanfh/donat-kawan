# Donat-KaWan

Project Tubes Web Pemesanan Donat

## Deskripsi

Aplikasi pemesanan donat berbasis Laravel menggunakan konsep MVC.

## Struktur Folder

-   app/ → Logika backend (Model, Controller, Filament)
-   resources/ → Tampilan frontend (Blade)
-   routes/ → Routing URL aplikasi
-   database/ → Migration database
-   storage/ → Penyimpanan gambar produk

## Bahasa yang digunakan

-   Laravel
-   MySQL
-   Filament
-   Tailwind CSS
